# Slate Web

Web client for [Slate](https://github.com/bulaya-ute/slate-server) — a self-hosted, Obsidian-compatible note platform. Point it at any Slate server URL, log in, and work with your vaults: live-preview Markdown editing, wikilinks, tags, backlinks, graph view, full-text search, and real-time sync.

> **Status: pre-release, under active development.**

- **Design spec**: [slate-server/docs/specs](https://github.com/bulaya-ute/slate-server/tree/main/docs/specs)
- **Stack**: React 18 · TypeScript · Vite · Tailwind · CodeMirror 6
